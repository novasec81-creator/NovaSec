5 signes que votre site a été piraté

7 juillet 2026 — Catégorie : Sécurité

Vous avez un site WordPress ou un site statique pour votre entreprise. Un matin, vous remarquez un ralentissement, ou un visiteur vous signale une page étrange. Est-ce un piratage ? Voici les 5 signes à connaître.

1. Votre site est anormalement lent

Un site hébergé correctement ne devrait pas mettre 10 secondes à charger du jour au lendemain. Si votre site rame sans raison, il est possible qu'il exécute des scripts cachés (minage de cryptomonnaie, envoi de spams).

2. Des fichiers inconnus apparaissent

Dans votre FTP ou gestionnaire de fichiers, vous trouvez des fichiers .php, .js ou .txt que vous n'avez jamais créés. C'est souvent le signe d'un backdoor laissé par un attaquant pour garder l'accès.

3. Votre site redirige vers des pages suspectes

Vous tapez votre URL et vous arrivez sur un site de casino, de fausses publicités ou une page d'alerte antivirus frauduleuse. Les hackers redirigent le trafic pour gagner de l'argent.

Plus de 60 % des sites piratés le sont via un plugin WordPress obsolète ou une vulnérabilité connue non corrigée.

4. Google affiche un avertissement

Si vous voyez "Ce site peut avoir été piraté" ou "Deceptive site ahead" dans les résultats Google, c'est que votre site a été signalé comme dangereux. Il faut agir rapidement pour ne pas perdre votre référencement.

5. Des comptes utilisateurs inconnus apparaissent

Dans l'admin de votre site, des comptes "admin2", "support" ou "backup" sont présents sans que vous les ayez créés. Un attaquant les a ajoutés pour garder un accès permanent.

Que faire si vous êtes piraté ?

  • Coupez l'accès à votre site temporairement (page de maintenance)
  • Changez tous les mots de passe (admin, FTP, base de données)
  • Supprimez les fichiers et comptes suspects
  • Restaurez une sauvegarde propre si vous en avez une
  • Faites appel à un professionnel pour un audit complet
